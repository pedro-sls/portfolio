# Roadmap

## Fase 1: Fundação

- Criar repositório público.
- Definir branches principais.
- Adicionar documentação de governança.
- Criar base Next.js com TypeScript, Tailwind e ESLint.

## Fase 2: Primeira versão visual

- Criar layout base.
- Implementar hero com links sociais.
- Implementar seção sobre mim.
- Implementar stack principal.
- Implementar projetos em destaque.
- Implementar contato.

## Fase 3: Qualidade e publicação

- Revisar responsividade.
- Ajustar SEO e metadados.
- Adicionar Open Graph image.
- Configurar deploy na Vercel.
- Adicionar README com instruções de desenvolvimento.

## Fase 4: Evolução

- Adicionar página individual de projetos.
- Integrar métricas públicas do GitHub.
- Criar formulário de contato com webhook do n8n ou serviço equivalente.
- Adicionar blog ou seção de estudos, se fizer sentido.
